# テニス乱数表メーカー

コート数、参加人数、性別、開催時間を入力して、ダブルス用の乱数表を作成する静的Webアプリです。

## 使い方

1. `index.html` をブラウザで開きます。
2. コート数、参加人数、時間、1試合の目安時間を選びます。
3. 参加者の名前と性別を入力します。
4. 「乱数表を作成」を押します。

## 特徴

- 参加者は最大24人まで入力できます。
- コート数は1〜4面、時間は2〜8時間から選べます。
- 参加人数はリストから選べます。コート数を変えると、1面4人、2面8人、3面12人、4面16人が初期選択されます。
- プレー回数が少ない人を優先して選び、なるべく全員が同じくらいプレーできるようにします。
- 同じペアや同じ対戦相手が続きすぎないように調整します。
- 男ダブ、女ダブ、ミックスの対戦タイプを色分けして表示します。
- 男ダブ・女ダブ多め、ミックス多めを選べます。
- 男ダブ・女ダブ、連続休憩回避、ゲーム数均等の優先順位を設定できます。
- 固定ペアを指定できます。固定ペアが多い場合は、後半からペアを崩す設定もできます。
- 対抗戦モードではチーム名とA/Bそれぞれの人数を設定できます。
- 対抗戦モードの参加者入力は、AチームとBチームを分けた縦一覧で入力できます。
- 試合結果を入力すると、個人ランキングと対抗戦のチーム勝敗を確認できます。
- 入力内容はブラウザの `localStorage` に保存されます。
- PWAに対応しているため、Android Chromeなどからホーム画面に追加できます。
- Service Workerにより、初回表示後は基本ファイルをオフラインでも読み込めます。

## 補足

このアプリは外部ライブラリを使っていないため、npm install やビルドは不要です。

PWAとして使う場合は、GitHub Pagesなどの `https://` URLで開いてください。`file://` で直接開いた場合、Service Workerは動作しません。
